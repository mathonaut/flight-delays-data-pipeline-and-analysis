# Mnemônicos

Este documento é um apêndice do dicionário de dados e define os padrões de nomenclatura e o significado dos atributos utilizadas no Data Warehouse do projeto.

## 1. Nomenclaturas e Significados

| Mnemônico | Significado                 |
| :-------- | :-------------------------- |
| `srk`     | Identificador               |
| `dim`     | Dimensão                    |
| `pk`      | Chave Primária              |
| `fk`      | Chave Estrangeira           |
| `idx`     | Índice                      |
| `fat`     | Fato                        |
| `air`     | Airline (Companhia Aérea)   |
| `apt`     | Airport (Aeroporto)         |
| `flt`     | Flight (Voo)                |
| `iat`     | IATA Code                   |
| `nam`     | Name (Nome)                 |
| `ste`     | State (Estado)              |
| `cod`     | Code (Código)               |
| `cty`     | City (Cidade)               |
| `val`     | Value (Valor)               |
| `lat`     | Latitude                    |
| `lon`     | Longitude                   |
| `dat`     | Data                        |
| `ful`     | Full (Inteiro)              |
| `yer`     | Year (Ano)                  |
| `mth`     | Month (Mês)                 |
| `day`     | Day (Dia)                   |
| `dow`     | Day of Week (Dia da semana) |
| `qtr`     | Quarter (Trimestre)         |
| `flg`     | Flag                        |
| `hol`     | Holiday (Feriado)           |
| `ori`     | Origem                      |
| `des`     | Destiny (Destino)           |
| `dep`     | Departure (Partida)         |
| `tme`     | Time (Tempo)                |
| `sch`     | Schedule (Agendado)         |
| `arr`     | Arrive (Chegada)            |
| `dis`     | Distance (Distância)        |
| `air`     | Air (Aéreo)                 |
| `elp`     | Elapsed (Decorrido)         |
| `dly`     | Delay (Atraso)              |
| `sys`     | System (Sistema)            |
| `sec`     | Security (Segurança)        |
| `acf`     | Aircraft (Aeronave)         |
| `wea`     | Weather (Clima)             |
| `ovn`     | Overnight (Durante a Noite) |
